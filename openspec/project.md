# Project Specification: NoteCascade

## Overview
NoteCascade is a visual MIDI keyboard practice application designed to help users master their piano skills through a game-like interface. It features a waterfall-style note display, real-time scoring, and an achievement system.

## Core Features
- **Real-time MIDI Input**: Connect physical MIDI keyboards via Web MIDI API.
- **Visual Feedback**: Falling notes synchronized with the selected song.
- **Scoring System**: Perfect, Good, Miss, and Wrong hit detection.
- **Song Library**: Built-in songs with varying difficulty levels.
- **Achievements**: Unlockable rewards for practice milestones.
- **Audio Feedback**: High-quality synthesizer sounds for played notes.

## Technical Stack
- **Framework**: Next.js 15+ (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Animation**: Framer Motion (motion/react)
- **Audio**: Tone.js
- **State Management**: Zustand
- **Icons**: Lucide React
