# NoteCascade API & Integration

## External Services
- **Firebase Auth**: User authentication.
- **Firebase Firestore**: Storing user scores and achievements.
- **Gemini API**: AI-powered music analysis and feedback (planned).

## Internal Hooks
- `useMidi`: Handles MIDI input and parsing.
- `useGameLogic`: Core game loop and state.
- `useAppStore`: Global application state.
